Project detail

This is a second year project that I made with the focus to deploy on mobile devices.

This project is also my first project for university that I need to design and develop from the ground up. I were responsible for managing as well as developing the project, therfore, I am very satisfied with it since working on this project has helped me to learn a lot about Unity and project management.

With regard to the development aspect of the project, I managed to implement multiple complex systems that taught me a lot in temrs of programming. The complex systems include:

  • Procedural level generation.
  • Inventory system.
  • Crafting system using Scriptable Objects.
  • Mobile input and optimization.
