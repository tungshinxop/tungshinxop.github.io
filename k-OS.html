Project detail

For the 7-day Brackeys Game Jam, I joined 10 other people to create a top-down shooter with the theme "Chaos".

I took the roles of Senior Programmer and Game Designer for this project. This is is my first time ever work in a team scale like this, so there were some problems occured, but I am still very happy with the final product and I learnt a lot working with the team.

Some of my noticeable contributions are the player controller and animation system. Since our project is using single frame animation, I needed to adapt it and write a system so that we can quickly test out the animations without having to modfiy them inside the Unity's animation window.
