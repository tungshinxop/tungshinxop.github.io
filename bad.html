Project B.A.D | University Project

Project information

  • Category: Programming
  • Duration: 6 months
  • Team size: 1 person
  • Tools: Unity, C#, Blender
  • Project URL: EXE Download

Project detail

This is one of the big solo project that I made for the final year of University. The aim for this project is to level up my understanding of Unity and 3D game development.

The final project features:

  • Complex movement system using vector math and Physics.
  • Platformer-like controller using techniques such as Coyote Time and Jump Buffering.
  • Weapon system using scriptable objects.
  • Cheat console.
  • AI using Finite State Machine system.
  • Dynamic weapon wheel.
  • Procedural animation using Inverse Kinematic.
  • Self-made object pooling and camera shake system.

Although this is not my first 3D project made in Unity, but it is one of my first 3D projects that is completed to a high standard. Thanks to working on this project, I managed to learn a tons. The new skills that I acquired include:

  • Exporting 3D models from Blender to Unity.
  • Working with Animator Override Controller.
  • Creating VFX using Unity VFX Graph.
  • Grey-boxing level using Unity's ProBuilder.


Regarding programming, I were in charged of making and monitoring complex systems such as dungeon genartions, weapon attachments, and weapon system. I am mostly satisfied that I managed to create a dungeon generation system using the drunken walk algorithm. This while is a pretty complex system, but by successfully implemented it, I have improved the skills of algorithm application and problem solving.
